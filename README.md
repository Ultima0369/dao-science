# Project Dao.Science

> 道可道，非常道；名可名，非常名。 ——《道德经》

## 愿景
将东方第一人称心性实践方法论，与当代认知科学、神经科学范式对齐，构建一份可被反复检验的心智操作系统手册。

## 核心公理
- **道** = 思维意识启用的动态过程（道 ≡ -∇_π G(π)）
- **一** = 无阻塞的心胸与觉知带宽（AB(t) = C_max - R_DMN(t)）
- **心智内容百分百是万物之相，非万物全部**
- **L0-L7 频谱** = 从物自体（L0）到关系坍缩（L7）的认知地形图

## 双螺旋架构
| 螺旋 | 定位 | 格式 | 产出 |
|------|------|------|------|
| 1. GitHub 开源仓库 | 可执行的技术手册 | Markdown + Mermaid 图表 | 22 个模块 |
| 2. 学术预印本 | 严谨的可引用论文 | LaTeX / PDF | 6 篇预印本 |
