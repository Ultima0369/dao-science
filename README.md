# Project Dao.Science

> 道可道，非常道；名可名，非常名。 ——《道德经》

## 愿景
将东方第一人称心性实践方法论，与当代认知科学、神经科学范式对齐，构建一份可被反复检验的心智操作系统手册。

## 核心公理
- **道** = 思维意识启用的动态过程
- **一** = 无阻塞的心胸与觉知带宽
- **心智内容百分百是万物之相，非万物全部**
