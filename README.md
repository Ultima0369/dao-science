# Project Dao.Science

> 道可道，非常道；名可名，非常名。 ——《道德经》

## 定位

道科学不是反科学，不是超科学，而是为科学本身提供关于"第一人称心智"的方法论支撑。它不是与物理学、生物学并列的另一门科学，而是一种供所有科学家使用的**元工具**：一种关于"如何优化观察者本身"的科学。

详见 [`POSITIONING.md`](POSITIONING.md)。

## 愿景
将东方第一人称心性实践方法论，与当代认知科学、神经科学范式对齐，构建一份可被反复检验的心智操作系统手册。

## 核心公理
- **道** = 思维意识启用的动态过程（道 ≡ -∇_θ G(π_θ)；离散形式：π_顺道 = argmin_π G(π)）
- **一** = 无阻塞的心胸与觉知带宽（AB(t) = 1 - [R_DMN(t) - R_0] / [R_max - R_0]）
- **心智内容百分百是万物之相，非万物全部**
- **L0-L7 频谱** = 从物自体（L0）到关系坍缩（L7）的认知地形图
- **涌现性** = 从部分之和到层级跃迁的系统性质生成
- **偏离代价** = Cost(π_dev) = G(π_actual) − G(π_opt)：偏离道的代价函数与预警信号
