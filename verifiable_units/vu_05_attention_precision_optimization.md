## 1. 形式化声明

### 1.1 核心命题

注意力在预测编码框架中等价于对预测误差**精度（precision）**的优化。元参数 α 控制注意力在「焦点聚焦」（高 α）与「全局觉知」（低 α）之间的分布：

- **高 α → 焦点注意**：精度权重集中在最可靠的少数通道，形成「赢者通吃」。
- **低 α → 全局觉知**：精度权重接近均匀分布，系统同时监测多个信息源。
- **灵活的 α**：系统能够根据任务需求与环境可靠性，在两者之间动态切换。

### 1.2 数学对象与定义域

设有 $N$ 个感觉通道，每个通道提供对隐状态 $s(t)$ 的带噪观测：

$$o_i(t) = s(t) + \eta_i(t), \quad \eta_i \sim \mathcal{N}(0, \sigma_i^2)$$

通道精度为 $\gamma_i = 1/\sigma_i^2$。在预测编码中，最优精度权重满足：

$$w_i^* \propto \gamma_i = \frac{1}{\sigma_i^2}$$

本项目提出的 α 控制插值：

$$\mathbf{w}(\alpha) = \alpha \cdot \mathrm{softmax}(\beta \, \boldsymbol{\gamma}) + (1 - \alpha) \cdot \frac{1}{N}\mathbf{1}$$

其中：
- $\alpha \in [0, 1]$：焦点-全局元参数
- $\beta$：softmax 逆温度，控制焦点模式的选择锐度
- $\boldsymbol{\gamma}$：通道精度向量
- $\mathbf{1}$：全 1 向量

隐状态估计为加权平均：

$$\hat{s}(t) = \sum_i w_i \, o_i(t)$$

### 1.3 边界条件

- 通道噪声已知或可在线估计
- 隐状态在观测时间尺度上相对稳定
- 注意力权重满足 $\sum_i w_i = 1$

## 2. 仿真脚本

### 2.1 文件位置

`simulations/attention_precision_optimization.py`

### 2.2 功能

- 对固定多通道系统，展示不同 α 下的精度权重分布
- 与贝叶斯最优权重 $w_i^* \propto 1/\sigma_i^2$ 对比
- 扫描 α，展示估计误差随焦点程度单调下降
- 模拟通道可靠性随时间变化，比较刚性 α 与自适应 α 的跟踪表现
- 模拟单一通道出现异常大偏差（attention hijack），展示灵活精度系统如何恢复

## 3. 预期结果与解读

### 3.1 静态权重分配

- α = 0：所有通道权重 ≈ 0.2，全局监测
- α 增大：权重逐渐偏向低噪声通道（ch1、ch2）
- α = 1：权重接近 softmax，形成对 ch1 的强烈偏向

**关键特征**：α 连续调节精度分布的焦点程度。

### 3.2 估计误差随 α 变化

- α 越低，估计越嘈杂（RMSE 高）
- α 越高，估计越接近贝叶斯最优（RMSE 低）
- 但完全聚焦也意味着对单一通道异常敏感，见病理演示

**关键特征**：存在一个与任务噪声结构相关的最优 α。

### 3.3 动态切换

当环境可靠性在半程反转时：
- 固定高 α 仍可能表现较好，如果它恰好适配新的可靠性排序
- 固定低 α 始终嘈杂
- 自适应 α（根据近期预测误差在线估计精度）能够持续接近最优

**关键特征**：灵活的精度估计比僵化的 α 更能适应变化。

### 3.4 注意力劫持与恢复

当 ch1 在 t=100..120 出现 +6 的大偏差：
- 刚性聚焦（α=1）被劫持，估计值被大幅拉偏
- 刚性开放（α=0）噪声大但不被劫持
- 灵活系统根据近期预测误差下调 ch1 精度，较快恢复

**关键特征**：「收放自如」不是永远高 α，而是能根据通道可靠性动态调整。

## 4. 与项目理论的对应

| 项目概念 | 仿真对应 |
|---|---|
| 注意力 = 精度优化 | 权重 $w_i$ 由通道精度 $\gamma_i$ 驱动 |
| α 参数 | 插值于均匀分布与 softmax 焦点分布之间 |
| 收（聚焦） | α → 1，赢者通吃 |
| 放（全局） | α → 0，均匀监测 |
| 自如（灵活） | 根据近期预测误差在线更新精度 |
| 怕/想要劫持 | 单一通道精度被异常上调，系统被锁定 |
| 痴 | α 失去动态范围，无法根据环境调整 |

## 5. 实验/经验对应（草案）

### 5.1 神经科学预测

1. **精度调控轴**：前扣带回（ACC）→ 蓝斑（LC）→ 丘脑网状核（TRN）的活动模式应随任务焦点需求变化。
2. **冥想训练**：长期冥想者应在焦点-开放切换任务中表现出更快的 α 切换速度和更大的 DMN-TPN 反相关变化幅度。
3. **ADHD**：可建模为 α 调控噪声过大或切换成本过高。
4. **焦虑/抑郁**：可建模为 α 僵化于高焦点（焦虑反刍）或低焦点（抑郁淡漠）。

### 5.2 可检验行为预测

- 在多重感觉线索任务中，被试应更多依赖低噪声通道，权重近似于逆方差。
- 当某通道突然变得不可靠时，健康被试应能迅速降低其权重；焦虑被试可能权重下降更慢。

---

## 6. 反事实条件

以下证据会削弱或推翻本命题：

1. 在多通道感知任务中，人类注意力分配与通道精度无关
2. 提高某通道精度不会增加其被注意的概率
3. 冥想训练不改变焦点-全局切换能力
4. 神经系统不存在与精度加权对应的神经增益调控机制
