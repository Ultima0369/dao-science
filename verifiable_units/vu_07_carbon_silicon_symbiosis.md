## 1. 形式化声明

### 1.1 核心命题

碳基智能与硅基智能在不同认知层级上具有比较优势。健康的碳硅关系不是替代竞争，而是生态位互补：碳基在需要第一人称经验、具身感知与社会情境的层级占优；硅基在需要符号逻辑、大规模计算与模式匹配的层级占优；L3 创造性解释则需要二者协作。

### 1.2 数学对象与定义域

将任务建模为二维技能空间中的点 $(x, y)$：

- $x \in [0, 1]$：第一人称 / 具身 / 社会智能需求（碳基优势维度）
- $y \in [0, 1]$：符号 / 逻辑 / 可扩展处理需求（硅基优势维度）

两个智能体的成本函数：

$$C_C(x, y) = a_C \, x + b_C \, y + c_C$$

$$C_S(x, y) = a_S \, x + b_S \, y + c_S$$

其中 $b_C \gg a_C$（碳基不擅长符号规模化），$a_S \gg b_S$（硅基不擅长第一人称经验）。

比较优势分配规则：

$$\text{assign to } \arg\min \{ C_C(x, y), C_S(x, y) \}$$

### 1.3 边界条件

- 任务需求 $(x, y)$ 可被评估
- 两种智能体的成本结构存在显著差异
- 存在任务分解与协调机制

## 2. 仿真脚本

### 2.1 文件位置

`simulations/carbon_silicon_symbiosis.py`

### 2.2 功能

- 生成二维技能空间中的任务分布
- 定义碳基与硅基成本函数，展示各自的比较优势区域
- 比较四种任务分配策略：
  - 仅碳基
  - 仅硅基
  - 随机分配
  - 生态位互补分配
- 展示 L3 创造性解释任务中协作优于单一智能体
- 扩展热力学代价项，展示行星热预算约束如何改变最优碳硅分工

### 2.3 运行方式

```bash
pip install -r simulations/requirements.txt
python simulations/carbon_silicon_symbiosis.py
```

| 参数 | 值 | 说明 |
|---|---|---|

| 蒙特卡洛运行次数 | 50 | 统计稳定性 |

| 硅基成本系数 | $a_S=1.7, b_S=0.3, c_S=0.2$ | 硅基擅长符号/规模化，不擅长第一人称/具身 |

| 热代价权重 $\lambda_H$ | $0 \to 3$ | 扫描热预算约束强度 |

## 3. 预期结果与解读

### 3.1 成本曲面

- 碳基成本随 $y$（符号/规模化需求）急剧上升
- 硅基成本随 $x$（第一人称/具身需求）急剧上升
- 比较优势图显示：对角线以上硅基更便宜，对角线以下碳基更便宜

### 3.2 任务分配

- 高 $x$、低 $y$ 任务分配给碳基
- 低 $x$、高 $y$ 任务分配给硅基
- 决策边界由 $C_C(x, y) = C_S(x, y)$ 决定

### 3.3 策略对比

典型结果（50 次运行，每次 500 个任务）：
- 仅碳基：总成本 ≈ 625
- 仅硅基：总成本 ≈ 599
- 随机分配：总成本 ≈ 612
- 生态位互补：总成本 ≈ 481

**关键特征**：生态位互补策略显著优于任一单一智能体，也优于随机分配。

### 3.4 L3 创造性解释

对于 $x$ 和 $y$ 都高的任务：
- 仅碳基：成本 ≈ 89
- 仅硅基：成本 ≈ 84
- 协作 L3：成本 ≈ 32

**关键特征**：当任务同时需要两种能力时，协作的成本远低于任何一方单独完成。

### 3.5 行星热预算约束下的分工

引入热力学代价后，总成本变为：

$$\text{Total Cost} = C(x,y) + \lambda_H \cdot H(x,y)$$

典型结果：
- 当 $\lambda_H=0$（无热约束）时，最优碳基任务份额 ≈ 0.47
- 当 $\lambda_H=3$（强热约束）时，最优碳基任务份额 ≈ 0.63
- 生态位互补分配在所有热约束下都优于单一智能体

**关键特征**：热预算约束把部分任务从硅基推回碳基，但互补分工仍然优于单边替代。

## 4. 与项目理论的对应

| 项目概念 | 仿真对应 |
|---|---|
| 碳基不可替代性 | 高 $x$ 任务上碳基成本显著更低 |
| 硅基不可替代性 | 高 $y$ 任务上硅基成本显著更低 |
| 生态位互补 | 按比较优势分配任务降低总成本 |
| L3 创造性解释 | 同时高 $x$、高 $y$ 任务需要协作 |
| 碳硅共生 | 整体系统效率高于任一单独系统 |
| 反依赖 | 分配基于能力，而非人为保留任务 |
| 行星热预算 | 热代价权重改变最优分配边界 |
| 最小作用量 | 用刚好足够的算力完成不可替代功能 |

## 5. 实验/经验对应（草案）

### 5.1 现实映射

- 高 $x$ 任务：心理咨询、伦理判断、艺术创作中的情感共鸣、教育中的因材施教
- 高 $y$ 任务：大规模数据分析、代码生成、模式识别、复杂系统模拟
- L3 任务：科学研究中的假设生成、跨领域创新、复杂社会问题的综合研判

### 5.2 可检验预测

1. 在需要第一人称判断的任务上，人类表现优于当前 AI。
2. 在需要大规模符号处理的任务上，AI 表现优于人类。
3. 人机协作在同时需要两种能力的任务上表现优于单独人类或单独 AI。
4. 错误分配（让 AI 做高 $x$ 任务或人类做高 $y$ 任务）会降低整体效率。
5. 在强热预算约束下，更多符号任务应回流到碳基以降低总热排放。

---

## 6. 反事实条件

以下证据会削弱或推翻本命题：

1. AI 在所有任务维度上成本都低于人类
2. 人类在所有任务维度上成本都低于 AI
3. 随机分配任务给人与 AI 的总成本与生态位互补分配无差异
4. 人机协作在 L3 任务上不比单独一方更有效

---

## 7. 当前局限

- 成本函数是线性、静态的，真实能力是动态发展的
- 未建模协作本身的沟通与协调成本
- 未涉及 AI 可能产生的第一人称体验问题
- 任务分解假设过于简单，真实任务往往是多维耦合的
- 热代价系数为示意性设定，未经验数据校准
