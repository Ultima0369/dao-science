## 1. 形式化声明

### 1.1 核心命题

涌现（emergence）与相变（phase transition）是两种不同种类的宏观秩序生成机制：

- **相变**：系统状态因外部控制参数越过阈值而发生跳变；微观单元的局部规则不变，只有全局统计量改变。
- **涌现**：系统的宏观性质来自内部组织关系；该性质在孤立单元中不存在，也不是由单一外部阈值触发的跳变。

### 1.2 数学对象与定义域

**相变原型**：2D Ising 模型

$$H = -J \sum_{\langle i,j \rangle} s_i s_j, \quad s_i \in \{-1, +1\}$$

控制参数：温度 $T$（或 $\beta = 1/T$）。
序参量：磁化强度 $|m| = |\frac{1}{N}\sum_i s_i|$。
临界温度（无限格点精确解）：

$$T_c = \frac{2}{\ln(1+\sqrt{2})} \approx 2.269$$

**涌现原型**：Boids 群集模型

每个个体的速度更新由三条局部规则合成：

1. **对齐**：$\vec{v}_i^{\text{align}} = \frac{1}{|N_i|}\sum_{j \in N_i} \vec{v}_j$
2. **聚合**：$\vec{v}_i^{\text{coh}} = \frac{1}{|N_i|}\sum_{j \in N_i} \vec{p}_j - \vec{p}_i$
3. **分离**：$\vec{v}_i^{\text{sep}} = -\sum_{j: d_{ij}<d_0} (\vec{p}_j - \vec{p}_i)$

其中 $N_i$ 为个体 $i$ 的感知邻域。控制参数：对齐权重 $w_{\text{align}}$。
序参量：极化度 $\phi = \|\frac{1}{N}\sum_i \hat{v}_i\| \in [0,1]$。

### 1.3 判别标准

| 特征 | 相变 | 涌现 |
|---|---|---|
| 驱动因素 | 外部控制参数越过阈值 | 内部局部规则持续作用 |
| 序参量曲线 | 在临界点处出现跳变/奇异性 | 随相互作用强度平滑增长 |
| 孤立单元 | 具有与整体相同的局部规则 | 不具有整体层面的性质 |
| 可还原性 | 原则上可还原为微观配分函数 | 宏观性质依赖关系结构，不可还原为单一个体 |
| 例子 | 铁磁-顺磁转变、水结冰 | 鸟群、蚁群、意识？ |

## 2. 仿真脚本

### 2.1 文件位置

`simulations/emergence_vs_phase_transition.py`

### 2.2 功能

- 对 2D Ising 模型进行 Metropolis 采样，绘制磁化强度随温度变化曲线
- 对 Boids 模型进行数值模拟，绘制极化度随对齐权重变化曲线
- 并排对比两种秩序生成机制的形态差异

### 2.3 运行方式

```bash
pip install -r simulations/requirements.txt
python simulations/emergence_vs_phase_transition.py
```

输出：
- `simulations/ising_phase_transition.png`
- `simulations/boids_emergence.png`
- `simulations/emergence_vs_phase_comparison.png`

### 2.4 关键参数

**Ising 模型**

| 参数 | 值 | 说明 |
|---|---|---|
| 格点大小 | $40 \times 40$ | 有限尺寸效应明显，但足以展示相变 |
| 平衡步数 | $10^5$ | Metropolis 热化 |
| 采样次数 | 30 | 对每个温度取平均 |
| 温度范围 | $T \in [1.5, 3.5]$ | 跨越理论临界温度 |
| 临界温度（无限格点） | $T_c \approx 2.269$ | 参考线 |

**Boids 模型**

| 参数 | 值 | 说明 |
|---|---|---|
| 个体数 | 200 | 足够展示群体行为 |
| 区域 | $60 \times 60$ | 周期性边界 |
| 感知半径 | 15 | 局部交互范围 |
| 分离半径 | 1.5 | 避免碰撞 |
| 最大速度 | 2.0 | 归一化单位 |
| 分离权重 | 0.1 | 弱分离，允许聚合 |

| 平衡步数 | 300 | 去掉瞬态 |
| 采样步数 | 300 | 平均极化度 |

## 3. 预期结果与解读

### 3.1 Ising 模型

- 低温区（$T < T_c$）：系统自发选择向上或向下磁化，$|m| > 0$
- 临界区（$T \approx T_c$）：有限尺寸下磁化强度快速下降
- 高温区（$T > T_c$）：热涨落破坏长程有序，$|m| \approx 0$

**关键特征**：在 $T_c$ 附近出现陡峭转变；转变由外部温度参数驱动。

### 3.2 Boids 模型

- $w_{\text{align}} = 0$：个体随机运动，极化度接近 0
- $w_{\text{align}}$ 增大：局部对齐产生 flock，极化度平滑上升
- $w_{\text{align}}$ 较大：形成稳定全局 flock，极化度趋于饱和

**关键特征**：极化度随内部交互参数连续增长；flock 作为整体性质不存在于孤立个体。

### 3.3 与项目主张的对应

| 项目主张 | 仿真对应 |
|---|---|
| 涌现 ≠ 相变 | 两条曲线的形态不同：一个在外部阈值处跳变，一个随内部交互强度平滑增长 |
| 涌现是关系结构的新性质 | Boids 的 flock 不能从单一个体的规则中读出 |
| 相变是参数驱动的状态翻转 | Ising 磁化在 $T_c$ 处翻转 |

## 4. 实验/经验对应（草案）

### 4.1 神经科学

- **相变样现象**：癫痫发作阈值、清醒-麻醉过渡（可能）
- **涌现样现象**：伽马同步、默认模式网络与执行控制网络的动态耦合模式

### 4.2 组织/社会系统

- **相变样现象**：市场在临界点处的崩盘、群体恐慌爆发
- **涌现样现象**：团队文化、创新氛围、信任网络

### 4.3 可检验预测

1. 真正的涌现性质应该满足：**移除或改变局部交互规则后，宏观性质消失或改变，而不仅仅是参数阈值移动**。
2. 相变性质应该满足：**在临界点附近，系统对相关微扰高度敏感，远离临界点时几乎完全由控制参数决定**。

## 5. 反事实条件

以下证据会削弱或推翻本命题：

1. 所有被归类为"涌现"的现象都被证明可还原为某个外部控制参数的相变
2. Boids 型模型的群集行为被证明只是隐藏相变的伪装（例如存在未被识别的序参量阈值）
3. 意识、文化等宏观性质被证明完全由底层物理参数的阈值决定
4. 复杂系统科学证明强涌现不存在，且弱涌现与相变在数学上等价

---

## 6. 当前局限

- 仿真为极简模型，真实系统通常是相变与涌现的混合
- Ising 模型在 $40 \times 40$ 格点上有明显有限尺寸噪声
- Boids 模型未与真实动物群集数据拟合
- "涌现"在哲学上仍有强/弱之争，本仿真仅展示弱涌现的定量特征
