## 1. 形式化声明

### 1.1 核心命题

关系调谐不是控制对方，而是调整自己的注意力精度分配，使对方的生命力张力能够自然释放与转化。两个 Agent 可通过耦合振荡器模型演示：孤立则各自耗散，刚性耦合则放大与捕获，临界承接式的调谐则实现「以柔克刚、刚柔并济」的共舞。

### 1.2 数学对象与定义域

设两个 Agent 的生命力张力为 $T_1(t)$、$T_2(t)$，遵循带外部冲动的衰减动力学：

$$\tau \frac{dT_i}{dt} = -T_i + I_i(t) + \text{support}_{ji}(t)$$

其中：
- $\tau$：张力自然衰减时间常数
- $I_i(t)$：周期性冲动输入
- $\text{support}_{ji}(t) = \kappa \cdot A_j(t) \cdot T_j(t)$：Agent $j$ 在 Agent $i$ 处于临界承接点 $t^*$ 时提供的支持

临界承接信号：

$$h(T, \dot{T}) = \sigma\big(T \cdot (-\dot{T}) - \theta\big)$$

当对方张力高且由盛转疲（$\dot{T}<0$）时，$h$ 达到峰值，触发承接式注意。

注意力函数：

$$A_j(t) = h(T_i, \dot{T}_i) \cdot \big(1 - \sigma(T_j - T_{\text{max}})\big)$$

即：看见对方的 $t^*$，同时不被自身过载淹没。

### 1.3 边界条件

- 张力可观测（至少概率估计）
- 冲动具有有限持续时间和疲劳拐点
- 注意力资源有限，不能被双方同时耗尽

## 2. 仿真脚本

### 2.1 文件位置

`simulations/relational_attunement_oscillator.py`

### 2.2 功能

- 生成两个具有相位偏移冲动序列的 Agent
- 比较三种关系模式：
  - **孤立**：无耦合，各自独立振荡
  - **刚性支持**：恒定高注意力，持续耦合
  - **调谐式承接**：仅在对方 $t^*$ 时提供支持
- 绘制张力时序、注意力脉冲、相位肖像
- 计算并对比平均负荷、峰值、自由能代理、组合张力方差

### 2.3 运行方式

```bash
pip install -r simulations/requirements.txt
python simulations/relational_attunement_oscillator.py
```

输出：
- `simulations/relational_attunement_timeseries.png`
- `simulations/relational_attunement_phase_portrait.png`
- `simulations/relational_attunement_metrics.png`

### 2.4 关键参数

| 参数 | 值 | 说明 |
|---|---|---|
| 衰减时间常数 $\tau$ | 4.0 s | 张力自然回落速度 |
| 冲动周期 | 14.0 s | 每个 Agent 的冲动间隔 |
| 冲动相位差 | 5.0 s | 模拟不同生命节律 |
| 冲动幅度 | 3.0 | 张力输入强度 |
| 调谐耦合强度 $\kappa$ | 1.5 | 承接支持的增益 |
| 刚性注意力水平 | 0.35 | 恒定耦合强度 |
| 承接阈值 $\theta$ | 0.5 | 触发 $t^*$ 检测的灵敏度 |

## 3. 预期结果与解读

### 3.1 张力时序

- **孤立**：两个 Agent 按各自节律涨落，低谷时缺乏支持。
- **刚性支持**：恒定耦合导致相位锁定与能量放大，峰值明显升高。
- **调谐承接**：注意力脉冲在对方 $t^*$ 处精准出现，张力曲线被「托住」但不放大。

### 3.2 相位肖像

- **孤立**：闭合但不规则的极限环，双方独立。
- **刚性支持**：轨迹向外扩张，显示耦合过度。
- **调谐承接**：轨迹紧致、有节律，体现共舞的秩序感。

### 3.3 指标对比

典型结果：

| 模式 | 平均张力 | 峰值张力 | 自由能代理 | 组合张力方差 |
|---|---|---|---|---|
| 孤立 | ~1.9 | ~1.8 | ~2.3 | ~0.22 |
| 刚性 | ~3.7 | ~2.9 | ~7.6 | ~0.69 |
| 调谐 | ~2.0 | ~1.8 | ~2.5 | ~0.23 |

**关键特征**：调谐式关系在几乎不增加个体负荷的前提下，实现了比孤立更协调、比刚性更节能的互动。

## 4. 与项目理论的对应

| 项目概念 | 仿真对应 |
|---|---|
| 以柔克刚 | 在对方冲动高点以注意力精度承接，而非对抗 |
| 临界承接 $t^*$ | 注意力脉冲在 $h(T, \dot{T})$ 峰值处触发 |
| 刚柔并济 | 注意力 $A(t)$ 在焦点与全局之间动态切换 |
| 被看见 | 对方张力因承接而平滑释放 |
| 双螺旋上升 | 调谐模式降低系统自由能而不消灭个体差异 |
| 关系调谐 | 耦合强度与时机共同决定关系质量 |

## 5. 实验/经验对应（草案）

### 5.1 人际互动

- 伴侣冲突中，一方情绪高峰时另一方的稳定在场可降低冲突升级概率。
- 管理者在团队高亢时「放」，在团队疲软时「收」。
- 咨询师对来访者的防御性高点保持非侵入式关注。

### 5.2 可检验预测

1. 在受控双人任务中，当一方在另一方冲动峰值后承接主导权，双方心率/皮肤电导的同步性更高且不过载。
2. 恒定高支持（刚性）会导致依赖或反抗；调谐式支持促进自主调节。
3. 关系质量与双方能否识别彼此的 $t^*$ 正相关。

---

## 6. 反事实条件

以下证据会削弱或推翻本命题：

1. 恒定高耦合始终优于或等于调谐式耦合
2. 人际冲突中，忽视对方的 $t^*$ 比精准承接更能降低张力
3. 关系调谐的效果完全由语言内容解释，与注意力时机无关
4. 双人系统中不存在可被检测的临界承接点

## 7. 当前局限

- 张力 $T(t)$ 是抽象变量，未直接对应具体生理指标
- 承接函数为启发式，未用真实人际数据校准
- 仅模拟两个 Agent，未扩展到多方关系网络
- 冲动输入为周期性高斯脉冲，真实人际冲动更不规则
